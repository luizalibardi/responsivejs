Smooth Scrolling Effect

Single-page websites are becoming a trend among web designers, both for their own projects and for client sites. There are a lot of cases out there in which a single-page site makes a lot of sense: if there is not a lot of content; if the content is all closely related; or in cases where a particular stylistic element works best on a single page.

In many cases, this arrangement represents the relationship between different sections of information and ideas better than the traditional multi-page model. You've likely seen it while browsing the web.

To provide navigation from section to section on this type of longer page, links are used â€“ just as with the more traditional pages. The difference is that links do not direct from one page to another; instead, they link to different points on the single page.

Linking to specific points on a page

When we want to link to a specific point on a page, the standard tool that HTML used to provide was a named anchor. At the destination point (e.g. a Contact form at the bottom of the page), we add the name attribute and a value to the anchor tag at the start of the section. This was a good way to target a place in the page, but it only worked with anchor tags. With HTLM5 the same thing can be done with the id attribute, that can be added to any tag, like a div tag:

   <div id="contact">Here goes the content</div>

To link to this point (e.g. from the top of the page), we link using the hash character (#) and the name of the destination id:

   <a href="#contact">Go To Contact Form</a>

Now let's add some smooth scrolling like the one seen in this example in three easy steps.

First: update you links

First, identify each of the anchor links in your page that you want to use smooth scrolling. On each link, add the smoothScroll class like this:

   <a href="#contact" class="smoothScroll">Go To Contact Form</a>

Second: Get the Scripts

Download this .zip file and unzip it. Add the tiny smoothscroll.js file to your site's scripts folder.

Last: Add The Scripts To Your Page

Now, it's time to add the script to your page. The script requires jQuery, so you'll need to add it as well. Add the following before the closing </body> tag: 

 <script src="http://code.jquery.com/jquery-1.11.0.min.js"></script>
 <script src="yourfolder/smoothscroll.js"></script>
