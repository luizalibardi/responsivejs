Toggling elements visibility

One piece of javascript code that I find useful is a function that toggles the visibility of an element. So when you click on an element it show and/or hides a specific element on the page. There are several ways of doing this with Javascript. Bellow are two them.

jQuery hide() and show()

With jQuery, you can hide and show HTML elements with the hide() and show() methods:

Content to hide

And here is the code:

 $("#hide").click(function(){
  $("p").hide();
 });
 
 $("#show").click(function(){
  $("p").show();
 });

You can also specify the speed of the hiding/showing by adding the optional speed parameter. The following example demonstrates the speed parameter with hide():

Content to hide

The speed can take the following values: "slow", "fast", or milliseconds. In the example bellow I used 1000 milliseconds.

 $("button").click(function(){
  $("p").hide(1000);
 });

jQuery toggle()

With jQuery, you can toggle between the hide() and show() methods with the toggle() method.

Shown elements are hidden and hidden elements are shown with the same button:

Content to hide

Bellow is the function for that. Notice that toggle() also accepts the speed parameter.

 $("button").click(function(){
  $("p").toggle();
 });
